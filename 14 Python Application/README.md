# Example of a Python Application

This is for my learning purpose.

## How to Use

1. Build the image: `$ docker build -t python-app:1.0 .`
2. Run a container of the image: `$ docker run python-app:1.0`

## Features

* Add a dependency.
* Pass an environment variable.
* Read from STDIN.
